# Limpeza-Perfil
Script criado para automatizar o processo de limpeza de perfis e chave de registro em Sistemas Operacionais Windows

Script para limpeza de perfis de usuario no Windows

Remove:
- Perfis registrados no sistema
- Pastas orfas em C:\Users

Uso:
Executar como administrador

Atencao:
Revise a lista antes de confirmar a exclusao
